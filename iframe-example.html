노션 계산기 위젯 iframe 설정

1. 기본 iframe 코드

노션에서 /embed 명령어 사용 후 다음 URL을 입력하세요:

https://your-domain.com/calculator-widget.html

2. iframe 직접 삽입 코드

다른 웹사이트에 삽입할 때 사용하는 코드:

<iframe 
    src="calculator-widget.html"
    width="360"
    height="600"
    frameborder="0"
    style="border: none; border-radius: 20px; box-shadow: 0 4px 20px rgba(0,0,0,0.1);"
    title="계산기 위젯">
</iframe>

3. 반응형 iframe 코드

크기가 자동으로 조절되는 반응형 코드:

<div style="position: relative; width: 100%; padding-bottom: 100%; overflow: hidden;">
    <iframe 
        src="calculator-widget.html"
        style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;"
        title="계산기 위젯">
    </iframe>
</div>

5. 노션 삽입 방법

  1. HTML 파일을 웹 호스팅 서비스에 업로드 (예: GitHub Pages, Netlify, Vercel 등)
  2. 노션 페이지에서 "/" 입력
  3. "embed" 선택
  4. 호스팅된 URL 입력
  5. 크기 조절 가능

6. 무료 호스팅 추천

  • GitHub Pages: github.io 도메인 제공
  • Netlify: 간단한 드래그 앤 드롭 업로드
  • Vercel: 빠른 배포와 좋은 성능
  • Surge.sh: 명령줄로 빠른 배포
